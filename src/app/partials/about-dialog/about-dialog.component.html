Project Dutch

I'm KhoPhi. I started this page to help sift through the Public Register Regular Labour and Highly Skilled Migrants website, which lists all the Dutch organizations that offer visa sponsorship for jobs.

As a non-Dutch, it was hard for me to know which organizations are tech-related, thus apply for a role.

This application takes all the 11,000-ish organizations listed on the PRRLHSM site, and categorizes them in a way one can quickly list and search through.

The source code for this project is available on Github
